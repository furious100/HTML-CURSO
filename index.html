Tags

(Voltar)

As TAG's são uma maneira de informar ao navegador que tipo de estrutura está sendo construída, como parágrafos, imagens, links e etc

Atributos de Tags

(Voltar)

São todos as propriedades que um atributo pode ter Abaixo listamos algumas delas

  1. Id: Identificação da Tag
  2. Style: Formata uma tag com o comando de CSS
  3. Class: Padroniza uma TAG

Links

(Voltar)

Comando para abrir site ou arquivo dentro da pasta

Atributos -> Target -> _blank: Abre em outra aba

Atributos -> Target -> _self: Carrega na mesma página
